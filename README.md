# 狼黑工具 - EasyEDA Pro 增强工具

## 🌟 简介
**狼黑工具** 是一款为 **嘉立创EDA专业版** 设计的实用插件集合，旨在提升PCB与原理图设计效率，提供便捷的图像、BOM、网络、器件等批量处理功能。

---

## 📖 使用文档
每个功能均配有 **流程图** 与 **步骤说明** ，请查看 [使用文档 - Github](https://github.com/WOLF4096/easyeda_wolfblack_tool/blob/main/UserManual.md)  

---

## 📦 安装

### 方式一：从文件导入
1. 从以下地址下载插件文件：
   - [GitHub Releases](https://github.com/WOLF4096/easyeda_wolfblack_tool)
   - [嘉立创EDA扩展广场](https://ext.lceda.cn/item/darksteel/wolfblack-tool)
2. 在EDA中打开：
   - **V2**：设置 → 扩展 → 扩展管理器 → 导入扩展
   - **V3**：高级 → 扩展管理器 → 导入
3. 选择下载的插件文件完成导入

### 方式二：从扩展广场安装（仅V3版本）
1. 打开嘉立创EDA专业版
2. 进入 **高级 → 扩展管理器**
3. 搜索 **“狼黑工具”**
4. 点击 **安装**

## 🛠️ 已发布功能

| 功能 | 简介 |
|------|------|
| **导入图片** | 将PNG/JPG/BMP图片导入PCB图层，支持尺寸与DPI调整 |
| **导入二维码** | 生成/识别二维码，支持图片或矢量模式，可放置到PCB |
| **位号查重** | 快速检查并定位PCB中重复的器件位号 |
| **丝印代码转换** | 将R/C/L元件编码转换为易读的工程值（如“104” → “100nF”） |
| **清空多余属性** | 自动清理供应商为空的元件属性，保持设计整洁 |
| **导入BOM** | 从Excel粘贴BOM，批量更新器件属性（制造商、型号等） |
| **批量替换器件** | 通过C编号批量替换器件，统一物料规格 |
| **批量修改网络** | 批量更改引脚网络名称，支持Excel规则粘贴 |
| **放置器件** | 从PCB或网表导入器件并自动放置到原理图 |
| **放置导线** | 根据PCB/网表网络差异，自动为未连接引脚放置导线 |
